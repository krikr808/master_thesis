1. Run start_octave.bat and wait for octave to start.
2. Change the working director by cd:ing ("Change Directory") to this directory. You can use the commands ls ("List information about file(s)") and pwd ("Print Working Directory") to navigate.
3. Type tanh_k_by_k_op and press enter.

If the plots tex files are chosen to be compiled and an error occurs, that is probably because some of the parameters are incorrectly specified, or an undefined macro has been used.
